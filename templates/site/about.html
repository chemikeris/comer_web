COMER web server

COMER web server has been developed at the Department of Bioinformatics of the Institute of Biotechnology (Life Sciences Center, Vilnius University).

This website is free and open to all users and there is no login requirement.

Software and Data Availability

COMER web server is fully open source. The source code of the {{ 'chemikeris/comer_web'|github_link:"frontend web server" }} and the {{ 'minmarg/comer-ws-backend'|github_link:"backend computation server" }} is freely available.

The components are also available:

  • {{ "minmarg/comer2"|github_link:"COMER2" }}: remote homology search on GPUs
  • {{ "minmarg/cother"|github_link:"COTHER" }}: fast and accurate threading and homology search tool
  • {{ "minmarg/ropius0"|github_link:"ROPIUS0" }}: inter-residue distance map prediction

Links to COMER2 and COTHER profile databases are provided on their respective websites.

Cite us

If COMER web server is useful for your research, please cite our articles.

The remote homology search method COMER has been published in the following articles:

  • Margelevičius M. Bayesian nonparametrics in protein remote homology search. Bioinformatics. 2016;32(18):2744-52, DOI: 10.1093/bioinformatics/btw213, PMID: 27153649.
  • Margelevičius M. COMER2: GPU-accelerated sensitive and specific homology searches. Bioinformatics. 2020;36(11):3570-3572, DOI: 10.1093/bioinformatics/btaa185, PMID: 32167522.
