COMER web server

COMER web server has been developed at the Department of Bioinformatics of the Institute of Biotechnology (Life Sciences Center, Vilnius University).

This website is free and open to all users and there is no login requirement.

Browser compatibility

The COMER web server has been tested and functions correctly on the following browsers.

Software and Data Availability

COMER web server is fully open source. The source code of the {{ 'chemikeris/comer_web'|github_link:"frontend web server" }} and the {{ 'minmarg/comer-ws-backend'|github_link:"backend computation server" }} is freely available. The {{ "minmarg/comer2"|github_link:"COMER2" }} GPU-accelerated sensitive and specific protein remote homology search is open source too and is available for as stand-alone software for local use.

The component tools are open source too and available for download as stand-alone software for local use:

  • {{ "minmarg/ropius0"|github_link:"ROPIUS0" }}: inter-residue distance map prediction
