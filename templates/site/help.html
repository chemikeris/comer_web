Help

This is the help site for the COMER web server.

Query input

COMER web server accepts several types of input:

  • Single sequence input in FASTA or plain text formats;
  • Single multiple-sequence alignment in FASTA or Stockholm formats;
  • Multiple sequences input in FASTA format;
  • Multiple inputs format, mixed of sequences and sequence alignments in plain text, FASTA, or Stockholm formats, separated by lines containing "//".

Input is assigned automatically to one of the above types. For example, FASTA input having all sequences of the same length is considered to be a multiple sequence alingment, and if sequence lengths differ, it is further analyzed as input of multiple sequences.

Maximum number of input sequences/alignments/profiles is 100.
