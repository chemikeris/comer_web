Help

This is the help site for the COMER web server.

Query input

COMER web server accepts two types of input: sequence input and multiple sequence alignment input.

Sequence input field accepts one or more sequence inputs in plain text or FASTA formats.

Multiple sequence alignment input field accepts one or more sequence inputs in FASTA formats or Stockholm formats.
