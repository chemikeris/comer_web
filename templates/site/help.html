Help

This is the help site for the COMER web server.

Query input

COMER web server accepts several types of input:

  • Sequence input in FASTA or plain text formats;
  • Multiple-sequence alignment in FASTA, A3M or Stockholm formats;
  • COMER profile.

Input is assigned automatically to one of the above types. For example, FASTA input having all sequences of the same length is considered to be a multiple sequence alignment, and if sequence lengths differ, it is further analyzed as A3M format.

Multiple input queries can be provided, mixed of sequences and sequence alignments in all formats, separated by lines containing "//". If the input contains a single FASTA query, each sequence can be treated as a separate query, if the checkbox "In case of single FASTA query, treat each sequence as a separate query" is checked.

Maximum number of input sequences/alignments/profiles is 100.
