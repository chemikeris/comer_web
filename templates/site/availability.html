Software and data availability

COMER web server is fully open source. The source code will be available for both frontend web server and backend calculation server.

The components are also available:

  • COMER2: remote homology search on GPUs
  • COTHER: fast and accurate threading and homology search tool
  • ROPIUS0: inter-residue distance map prediction

Links to COMER2 and COTHER profile databases are provided on their respective websites.
