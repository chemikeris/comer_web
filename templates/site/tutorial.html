Tutorial for using the COMER web server

Users are encouraged to refer to the {% help_link %} to familiarize themselves with the services provided by the COMER web server.

The main page of the COMER web server is where input data can be entered. Enter queries in the text field. The separator line “//” indicates the end of an individual query, which can represent a sequence, plain or in FASTA format, an MSA in aligned FASTA, STOCKHOLM, or A3M format, or a COMER2{% comment %} or COTHER{% endcomment %} profile.
